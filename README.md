**SnackStack** is a Tinder-style recipe discovery app designed to make cooking fast, fun, and personalized.

- **Celiac-safe (gluten-free)**
- **Lactose-free**
- **Vegetarian**
- **Vegan**
- **Kosher**

Each recipe includes:

- Images  
- Ingredient breakdown
